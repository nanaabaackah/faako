# faako

SaaS platform for a configurable ERP plus a public website, signup funnel, and API.

Current temporary domain: faako.nanaabaackah.com

Stack:
- Frontend: React + Vite + React Router + Context + CSS.
- Backend: Netlify Functions (REST).
- Database: PostgreSQL (Railway).
- Schema/migrations: Prisma.
- Tooling: Playwright, ESLint, Netlify CLI.

Repository layout:
- apps/website: Marketing site and signup funnel.
- apps/erp: Tenant-facing ERP web app.
- apps/api: Netlify Functions REST API + Prisma schema.
- docs: Product, ops, and architecture notes.
- config: Shared configuration and templates.
